■ はじめに

ご利用頂きまして、誠にありがとうございます。
nuxt3で トースト機能を作成致しました。
front側とbackend側をあえて分けて作成しております。



■ 使用手順
clone後、package内容を読み込む必要がある為、
OSがwindowsの場合は「クローン後に実行.bat」ファイルの実行をお願いします。


1.　クローン後に実行.batを実行

2. 別コンソールを開き、下記を実行
cd front
npm run dev

3. 別コンソールを開き、下記を実行
cd backend
npm run dev
